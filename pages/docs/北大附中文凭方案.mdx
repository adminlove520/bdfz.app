# 北大附中文凭方案
培养目标是一所学校的根本方向，是学校价值观的明确表达。不同学校明确个性化的培养目标，和而不同，学生才可能有多元选择，并进而成就个性发展。
如上，是北大附中的培养目标。而北大附中文凭制，则是认定北大附中毕业生达到学校培养目标程度的制度。
对于完成国家规定学习内容的学生，根据国家规定，毕业时可获得高中毕业证书。对于符合与深度符合北大附中培养目标的学生，学校授予北大附中文凭。
北大附中文凭分为北大附中普通文凭和北大附中荣誉文凭两种：
北大附中普通文凭，简称北大附中文凭，是指获得学校规定学分并符合北大附中培养目标，经个人申请举证、学校认定后，由学校授予的文凭。
北大附中荣誉文凭是指在获得北大附中普通文凭基础上，在北大附中培养目标的某一方面有突出表现或成果；经个人申请举证、学校认定后，由学校授予的文凭。
## 2、北大附中文凭要求
2.1
完整接受北大附中高中部两年课程培养
2.2 规定学分
高中毕业学分基础上，必须包含的学分见北大附中课程手册，由学业管理办公室核定。
2.3 其他
授予文凭前，不得有未经消除的处分记录。如有，取消其文凭。
如有“记过”或“留校察看”处分记录，不得申请荣誉文凭。

## 普通文凭申请流程
点击申请库，依据模板完成申请文书
学业管理办公室：学分核查
书院督导办公室：处分核查
核查有问题的文档，将回复文档或单独文档通知，不予通过。
核查无问题，默认通过不予回复。
## 荣誉文凭申请流程
在2023届荣誉文凭申请库内依据模板完成申请文书
7.1 维度
已获得北大附中普通文凭
且平均学分积点在3.5+（如低于该积点，需在申请文书内专门解释）
且在书院内有明显贡献，或社团参与社团组织成就
在北大附中培养目标内，某一方面有突出表现或成果
7.2 流程
荣誉文凭委员会线上初审
委员会成员依据最终名额上限投票，记为最终结果50%
由书院议事会对本书院依据名额投票，记为最终结果50%
上述两者合并，按照票数排行，产生最终年度结果
荣誉文凭委员会成员：校长+书院督导组
8、授予文凭证书
所有文凭，均于高三毕业典礼时授予。
