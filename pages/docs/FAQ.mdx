# 学校可以点外卖吗
学校各类文件中目前没有明确规定学校是否可以点外卖。在疫情前学校曾允许点外卖，并设立了专用外卖柜

# 1.1 如何申请免体？
1. 有二甲等级以上医院诊断证明；
2. 在希悦系统应用中心找到【免体申请】，并提交材料；
3. 等待审批结果。

# 1.2 6选3换科流程
1. 在希悦系统应用中心找到【课程调整申请】，并根据要求提交材料；
2. 等待处理结果。
说明：换科申请书必须包含以下内容：
1. 调整的内容；
2. 换科的原因；
3. 换科所面临的风险；
4. 换科后的学习计划；
5. 相关人签字：本人、家长/监护人、导师、退出课程的任课老师、退出课程的课程首席、新加入科目的课程首席。（请注意为课程首席，无需找学科长签字）

# 2.1 学考合格考如何报名？
1. 市级学科合格考（语数外及高考6选3科目）：学生登陆北京教育考试院官网进行自主报名，学校不统一上报名单。报名期限一般为一周。具体考试安排以学业管理中心通知为准；
2. 区级技术合格考：由学校统一上报名单，默认未参加或未通过的学生都报名，具体考试安排以学业管理中心通知为准；
3. 高三艺术合格考：由学校根据学生高一高二艺术成绩统一录入分数，学生不需要参加考试；
4. 高三体育合格考：由学校统一上报名单，默认全体高三都参加考试，具体考试安排以学业管理中心通知或体育中心通知为准；

# 2.2 学考合格考缺考或不合格对个人是否有影响？
1. 高考6选3科目未合格，则无法参加高考；
2. 物理、化学、生物、政治、历史、地理、信息技术、通用技术、艺术、体育需全部合格，才能拿到北京市高中毕业证。

# 2.3 学考合格考是否能补考？
可以，毕业2年内均可以本校学生身份报名参加考试（2年之后按照社会考生身份参加考试）。


# 3.1 老师记错考勤了怎么办？
1. 找到任课老师；
2. 在希悦应用中心找到【学生考勤修改】；

3.填写相关信息，点击【提交】即可。

# 4.1 六选三选科变更后是否更可以更换教材？
1. 高二教材的征订时间为每年7月，7月之前更改了选科的同学，以最新选科领取教材；
2. 7月后更换选科的同学需要按照原选科领取教材，并自行购买所换学科的教材。

# 4.2 如果开学后没有领取到教材怎么办？
所有教材都已经发放至各个学科相应的班级中，如果未领取到可以先询问任课老师是否有剩余教材，如果没有可以到学业管理中心办公室找邓老师领取。

# 5.1 学校平板电脑使用过程中出现问题或需要申请APP该如何处理？
1. 在希悦系统应用中心【平板电脑使用问题反馈】中发起申请；
2. 等待审批。
# 5.2 平板电脑如何申请领取？
1. 打印 [平板申请承诺书](https://pkuschool.yuque.com/office/yuque/0/2023/pdf/22916111/1678863091715-15f9a9cc-cb80-42c0-a194-d3696b328317.pdf)
2. 本人和家长签字确认；
3. 将签完字的承诺书交至学业管理中心办公室找李志华老师处；
4. 领取平板电脑。
# 5.3 平板电脑相关问题和建议应该如何反馈？
可以发送邮件至学业管理中心办公室（xueye@pkuschool.edu.cn），或者企业微信李志华老师或邓慧翀老师进行反馈。
# 5.4 是否还可以申请个人设备在校使用？
个人设备申请只能一学年申请一次，可以申请时我们会通过希悦发送通知告知大家。
# 5.5 设备丢失怎么办？
个人平板电脑不慎丢失，需要购买相同配置及型号的设备，购买期间可先领取备用机

# 6.希悦、O365、语雀、企业微信等信息化系统
1. 在希悦应用中心找到【信息化系统问题反馈】提交问题反馈；
2. 等待问题解决。
说明：
1. 如果登录遇到问题，请说明需要登录的用户名和身份；
2. 如果系统有bug，请务必录下视频，反馈你的设备系统和具体问题，我们好进一步联系技术人员；
3. 如果涉及希悦系统无法登陆的问题，无法发起审批，可以发送邮件至学业管理中心办公室（xueye@pkuschool.edu.cn）反馈问题。
